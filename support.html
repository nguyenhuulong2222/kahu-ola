How to get help

Email long@kahuola.org with a short description of the issue, the page involved, and what you expected to happen.

For urgent incidents, follow official Maui emergency guidance first.

FAQ

  • Why do the module buttons say coming soon?

    The landing portal is a production-ready public front door. Interactive module actions remain intentionally disabled until each workflow is ready for safe release.

  • Does the status board make live network calls?

    No. The landing page uses a static demo status design and computes the "Last checked" time locally in the browser without calling external services.

  • Is Kahu Ola an official government application?

    No. Kahu Ola is an independent civic-tech project for Maui County and should be treated as a supplemental informational tool.

Troubleshooting checklist

  • Refresh the page to ensure the latest static build is loaded.
  • Confirm that ./styles.css is loading and that browser extensions are not blocking local assets.
  • Use the root production paths: /, /privacy.html, and /support.html.
  • If a module is labeled coming soon, that behavior is expected in the current release.

Permissions guidance

Location-related access should only be requested when a specific safety function needs it. The intended model is contextual access for a clear purpose, not continuous background collection.

Privacy reminder

Location data is processed via Edge Computing on the user's device (on-device) or an equivalent edge path; we do not store raw precise location on our servers by default.

This privacy posture is designed to support safety functionality while minimizing exposure of raw precise location data.
